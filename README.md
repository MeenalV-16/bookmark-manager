# Bookmark Manager

A simple bookmark manager that allows you to add, categorize, and tag websites. You can also sort and filter bookmarks by category and tag.

## Features
- Add and display bookmarks with title, URL, category, and tags.
- Sort bookmarks by category or tag.
- Simple, user-friendly interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Add a bookmark by filling in the URL, title, category, and tags.
2. Click "Add Bookmark" to save.
3. Use the dropdowns to sort or filter bookmarks by category and tag.
